Markdown 语言学习

2017-10-26
md, script

Markdown 的目标是实现「易读易写」。

Markdown 语法的目标是：成为一种适用于网络的书写语言。

Markdown 是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

Markdown 的语法简洁明了、学习容易，而且功能比纯文本更强。

建议使用缩进表示归属上一层级，如果出现块情况非空行不能处理归属问题
