HBuilder svn 插件安装

2017-10-28
mobile, plugin

安装步骤

  1. 工具 >> 插件安装
  2. 选择 SVN >> 安装，重启 HBuilder
  3. 工具 >> 插件安装 >> 手动安装插件（SVN 版本问题，必须更新版本）
  4. Work with： http://subclipse.tigris.org/update_1.10.x
  5. 全部保存，一直点击【下一步】，最后【完成】，重启 Hbuilder
  6. 视图 >> 定制透视图
  7. 命令和快捷方式选项卡勾选 SVN，tool bar 和 menu 勾选 SVN
  8. 确定后，安装完毕
